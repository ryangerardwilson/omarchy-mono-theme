# AGENTS.md

## Scope

This repo owns the `mono` Omarchy theme.

## Source Of Truth

- The theme lives at `~/.config/omarchy/themes/mono` and is the local source of
  truth.
- Make durable theme edits directly in this repo.

## Workflow

- Commit and push this repo before treating a change as the real published
  theme.
- To activate the theme locally, run `omarchy-theme-set mono`.

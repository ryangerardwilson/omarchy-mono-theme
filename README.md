# mono

A published Omarchy theme that inverts the monochrome dark variant into a pure
white light mode.

## Install

```bash
omarchy-theme-install https://github.com/ryangerardwilson/omarchy-mono-theme.git
```

If you clone it manually:

```bash
git clone https://github.com/ryangerardwilson/omarchy-mono-theme.git ~/.config/omarchy/themes/mono
omarchy-theme-set mono
```

## Included Overrides

- `colors.toml` for Omarchy-generated theme files
- `alacritty.toml`, `btop.theme`, `hyprland.conf`, `mako.ini`, and `tmux.conf`
  for the monochrome UI treatment
- `mono.tmTheme`, `neovim.lua`, and `light.mode` for editor and light-mode
  alignment
- `icons.theme` and `vscode.json` for app-specific theme selection
- `wallpaper.txt`, `wallpaper.svg`, `preview.png`, and `backgrounds/0-mono.png`
